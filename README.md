## ✨ Features
* **Modern Stack:** Next.js App Router for optimal SEO and performance.
* **Component Library:** Built with `shadcn/ui` for beautiful, editable components.
* **Payments:** Fully integrated with Stripe Checkout for seamless transactions.
* **State Management:** Uses Zustand for simple, fast cart state management.

## 🚀 Getting Started

**1. Clone and Install:**
```bash
git clone https://github.com/razinahmed/modern-ecommerce-storefront.git
cd modern-ecommerce-storefront
npm install
```

**2. Setup Stripe Keys:**
Create a `.env.local` and add:
```
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_...
STRIPE_SECRET_KEY=sk_test_...
```

**3. Run the Development Server:**
```bash
npm run dev
```

Browse to `http://localhost:3000` to see your storefront!
